Q. What is a version control system? A. Class of systems responsible for managing changes to computer programs, documents and large websites. Q. What is Git? A. distributed version control system that tracks changes in any set of computer files. Q. What is the difference between Git and Github/ A. Git is a control system used more for speed, data integrity and support for non-linear workflows.Github is used more for control, bug tracking, software feature requests and task management. Q. Who started Github and how was it started? A. It was developed by Chris wanstrath, P.J. Hyett, Tom Preston-Werner and Scott Chacon. It was developed to help them with their own work. Q. what company owns it now? A. It is owned by microsoft. Q. How much does a Github account cost? A. $49 per month. Q. What is the octocat? A. The mascot of the source-code hosting service Github.
